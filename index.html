Projects

Adiar

github.com/ssoelvsten/adiar

Following up on the work of Lars Arge in 1996, Adiar provides an easy-to-use implementation of Binary Decision Diagrams that make use of Time-Forward Processing to achieve efficient manipulation of BDDs, that far outgrow the memory limit of the given machine.

Cadiar

github.com/ssoelvsten/cadiar

A formal verification of the Adiar BDD package in the Isabelle proof assistant.
